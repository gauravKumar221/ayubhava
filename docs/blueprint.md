# **App Name**: Bit max

## Core Features:

- Dashboard Overview: Display key metrics like total patients, appointments, and revenue in an intuitive dashboard.
- Patient Management: Maintain a searchable patient database with key info and medical history.
- Payment Tracking: Track daily/monthly payments, invoice status, and total revenue with charts.
- Visitor Analytics: Provide insights into daily visitor trends with date-range filters.
- Doctor Schedule: Manage doctor availability, timings, and booked appointments.
- AI-Driven Report Generator: A tool to generate summaries and key points from unstructured notes (doctor notes, research papers, discharge summaries etc.)
- User Profile Management: Admin profile section for settings, password changes, and notifications.

## Style Guidelines:

- Primary color: Soft blue (#A0CFEC) to evoke a calm and professional medical environment.
- Background color: Very light blue (#F0F8FF) to maintain a clean and airy feel.
- Accent color: Light green (#90EE90) for highlights and call-to-action buttons.
- Body and headline font: 'Inter' sans-serif for clear and modern readability.
- Use line icons from a consistent set, related to hospital administration (medical symbol, user, file, payment, etc.).
- Modern dashboard layout with a sidebar for navigation and cards with rounded corners.
- Subtle transitions and hover effects on cards and buttons to improve UX.
